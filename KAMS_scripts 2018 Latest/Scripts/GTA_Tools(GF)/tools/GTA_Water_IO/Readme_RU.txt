

	GTA Water IO 1.0b
======================================

обновлено: 20.03.2019

Совместимость: 3ds max 2009 и выше (возможно версии и младше)

Автор: Goldfish 
Оф поддержка скрипта: https://vk.com/topic-99725313_35674835
Постоянная ссылка для скачивания/обновления скрипта: https://yadi.sk/d/NsuFPYkGNsuixA


		*****
Скрипт для экспорта и импорта воды для gta sa
имеется интерфейс для настойки параметров воды

особенности:
- поддержка экспорта треугольной воды
- импорт воды вместе с ее всеми параметрами
- вспомогательные утилиты для работы и настройки воды

в комплекте со скриптом имеется подробное описание интерфейса скрипта (в "интерфейс скрипта.jpg")


Установка:
Поместить скрипт GTAWaterIO.ms в любую папку, например в ../3dmax_folder/Scripts/
Запускать из меню программы 3dmax либо перетаскиванием в окно 3dmax
